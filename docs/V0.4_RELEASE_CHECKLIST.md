# v0.4 Release Checklist

## Automated checks

- [x] Shared extraction unit tests pass.
- [x] Destination-capacity enforcement is tested.
- [x] Individual unreadable files do not stop other copies.
- [x] Cancelled ZIP extraction removes the partial archive.
- [x] Progress events include files, bytes, and current path.
- [x] macOS SwiftUI target builds.
- [x] Universal self-contained Mac engine builds and runs.
- [x] Mac friend-test package builds and passes code-signature verification.
- [x] Approved icon assets are present for macOS, Windows, Linux, Microsoft
  Store packaging, and GitHub.

## Package validation

- [ ] GitHub Actions builds the Windows x64 package.
- [ ] GitHub Actions builds the Linux x64 package.
- [ ] Microsoft Store packaging job produces a v0.4 `.msixupload`.
- [ ] Current Mac package is notarized and the ticket is stapled.

## Clean-machine and tester validation

- [ ] Mac tester opens the app without installing Python.
- [ ] Windows tester opens the standalone executable.
- [ ] Linux tester opens the standalone executable.
- [ ] A non-technical tester previews, selects, extracts, and opens the report
  without separate instructions.
- [ ] Cancellation is tested only with disposable sample data.
- [ ] No source-drive writes are observed during Safe Rescue testing.

Do not tag the public v0.4 release until the package and clean-machine items are
complete.
