# Privacy Policy Draft

Drive Rescue Assistant is local-first.

Default privacy promise:

> Drive Rescue Assistant runs locally. It does not upload, transmit, sell, or analyze your files on remote servers. File names and recovery logs stay on your device unless you choose to export or share them.

## Data Handled

The tool may read:

- Drive names.
- Mount paths.
- File names and folder names.
- File sizes.
- Copy success and failure information.

The tool does not intentionally collect:

- File contents for analytics.
- Personal data for remote processing.
- Cloud account data.

## GitHub Repo Rule

Recovered files, personal scan reports, and logs containing private filenames should not be committed.
