# Drive Rescue Assistant - Tester Invitation

Hi,

I am testing an early version of **Drive Rescue Assistant**, a desktop app that helps people preview and copy readable files from external drives without deleting or changing the original files.

Please download the version for your computer:

## Mac

Download:

https://raw.githubusercontent.com/jibolufc/Drive-Rescue-Assistant/master/downloads/DriveRescueAssistant-mac-friend-test.zip

1. Download and unzip the file.
2. Open `DriveRescueAssistant.app`.
3. If macOS blocks it, Control-click the app, choose `Open`, and confirm.
4. The v0.4 Mac build is self-contained and does not require Python.

If you previously downloaded a copy that macOS described as damaged, delete that old copy and download the replacement from the same link again.

## Windows

Download:

https://github.com/jibolufc/Drive-Rescue-Assistant/releases/download/v0.2.2/DriveRescueAssistant-windows-x64.zip

1. Download and extract the ZIP file.
2. Open `DriveRescueAssistant.exe`.
3. If Microsoft Defender SmartScreen appears, choose `More info`, check that the app is named DriveRescueAssistant, then choose `Run anyway`.
4. The Windows version is self-contained and does not require Python.

## What To Test

Please use a drive containing files that are already backed up or are not irreplaceable.

1. Connect an external drive.
2. Open Drive Rescue Assistant and refresh the drive list.
3. Select the drive, or choose a source folder manually.
4. Choose a different drive or folder as the extraction destination.
5. Select a file type such as Documents or Photos.
6. Choose `Preview Files`.
7. Select or deselect individual files.
8. Optionally enable ZIP compression.
9. Extract the selected files and check that they open correctly at the destination.

The app should not delete, format, repair, or modify the source drive.
