# Drive Rescue Assistant Brand Assets

The approved mark combines a storage drive with an open chain. The open link
represents safe access and recovery; it is not a claim that the app decrypts,
unlocks, or bypasses protected data.

`DriveRescueAssistant-icon-1024.png` is the master raster asset. Platform
variants are generated from that file so the product keeps one consistent
identity across macOS, Windows, Linux, Microsoft Store, and GitHub.
